## Build the image
```bash
docker build -t <image-name> .
```

## Run image container
```bash
docker run -d --name <name-container> -p 80:80 <image-name>
```

---

## Example
### Build Image
```bash
docker build -t web-music .
```
### Run image container
```bash
docker run -d --name web-music -p 80:80 web-music
```
